# 🚀 Nova Strike

Nova Strike is a 2D space shooter game developed using Godot Engine.

## 🎮 Features

- Space shooter gameplay
- Player movement and shooting
- Enemy AI system
- Boss battles
- Power-ups
- Score system
- Pixel art graphics

## 🛠 Technologies

- Godot Engine 4
- GDScript
- 2D Game Development
- Object Oriented Programming

## 🎯 Learning

Built this project to understand:
- Game loops
- Scene management
- Collision detection
- Entity systems
- Game architecture
